Description

Bootstrap Dashboard Kit is the perfect starting point for your next web app built with Bootstrap 5. A free and minimal starter kit to build fast and modern dashboards and applications. bootstrap-dashboard-kit is a GitHub template repository for creating new Bootstrap-powered npm projects, maintained by taddart creators.

Repo Template

Setup as a starter template, you can easily generate a new GitHub repository. From the repository homepage, click the "Use this template" button.

11ty (Eleventy) Integration

Besides the static HTML version, it also includes a basic integration with 11ty: a simpler static site generator written in JavaScript that helps transform a directory of Data Discovery into HTML.

11ty is part of the Jamstack architecture and it enable sites and applications to be delivered with greater confidence and resilience than ever before.

What's included

  • Includes Eleventy static site generator for building the layouts and pages
  • Includes Bootstrap (currently using v5) source files via npm
  • Includes taddart CSS as our design system based on Bootstrap 5
  • Includes npm scripts for compiling and autoprefixing Sass, compiling JS, watching for changes, and starting a basic local server
  • Example stylesheet (scss/main.scss)
  • Example JavaScript file (js/main.js) showing how to import all Bootstrap
  • HTML, CSS, and JS minification
